Installing Defold

Defold is free so you can continue your projects at home. The installation files can be downloaded here: defold.com/download

In school, your network manager will need to:

  • Whitelist github.com and associated IP addresses to allow the download of templates.
  • Make sure java.exe is an exception from any network and classroom management software SSL interception.  Lua source code is compiled to Java bytecode by Defold.
  • Set an allow policy for “C:\Users\*\Appdata\Local\Defold\*” to ensure students are able to run the compiled code.
