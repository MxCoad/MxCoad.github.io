Common errors and how to fix them

  • Code I have written in a script isn't executing.
  • ERROR:GAMEOBJECT: The collection 'xxx' could not be created since there is already a socket with the same name.
  • Installing Defold on a school network.
  • Keyboard inputs are not detected when multiple keys are pressed at the same time
  • Run-time error: java.net.SocketTimeoutException: connect timed out.
  • Sprites are sliding down the screen or bouncing off each other.
  • Sprites are spawning from a factory in the wrong place.
  • Sprites have extra pixels when rendering.
