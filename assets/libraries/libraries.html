Libraries index

Libraries allow you to share code between projects and add additional functions beyond those included in Defold.

  • DefOS

Provides extra native OS functions including:

    • Setting/disabling window resizing.
    • Getting the window size and position.
    • Retrieving display information, including resolution.
    • Showing/hiding the mouse.
    • Handling cursors.
    • Retrieving the game path.
