Sprites index

It can be hard to find royalty-free images suitable for your projects online without spending a long time looking. These are some useful sprites collected for the tutorials.

  • Angband.zip - a collection of 32x32 dungeon crawler sprites suitable for a tile set.
  • Asteroids.zip - a collection of large, medium and small asteroids suitable for an animation set.

  • Breakout.zip - a collection of sprites suitable for block-hitting games.
  • Earth.zip - a collection of 40 sprites to create a rotating earth suitable for an animation set.
  • Explosions.zip - a collection of different explosion effects.

  • Hit marker.zip - an animation set for a hit marker.
  • Midway.zip - a collection of aircraft suitable for a 1942 game.
  • Misc.zip - additional images used in the tutorials.
  • Muzzleflash.zip - an animation set for a muzzle flash.
  • Platformer.zip - a collection of sprites suitable for simple platform games.
  • Player.zip - a collection of sprites suitable for learning about animation sets.
  • Rocket.zip - an animation set for a rocket.
  • Snake.zip - sprites for the classic character.

  • Spaceship.zip - a collection of small spaceship sprites.
  • Walls.zip - a collection of simple connecting wall sprites.
