Spotter | Creating a player target object and spawner.

When the player clicks on the picture, an object will be spawned at that location. By default it will be a picture of a cross to indicate the player was incorrect. If that object causes a collision with a difference object, it will change the picture shown to a tick. To spawn objects at run-time we need a factory, but first we need to create the object to be spawned. This will be known as the "target".

  1. In the assets panel, right click 'main' and select, 'New...', 'Game Object'.
  2. Give it the name, 'target' and click, 'Create Game Object'.

We are not creating this object in a collection because it doesn't belong to any particular level, we want to be able to spawn these objects on any level. Thinking ahead here allows us to create a reusable program component.

  1. In the outline panel, right click, 'Game Object' and select, 'Add Component', 'Sprite'.
  2. In the properties panel, change the 'Image' property to be, '/main/graphics.atlas'.
  3. In the properties panel, change the 'Default Animation' property to, 'cross'.
    Remember you can press F to zoom out.
  4. In the outline panel, right click, 'Game Object' and select, 'Add Component', 'Collision Object'.
  5. In the properties panel change the 'Type' property to 'Kinematic'.
  6. In the outline panel, right click, 'collisionobject' and select, 'Add Shape, 'Sphere'.
  7. In the editor panel, use the object controls to move and size the sphere to match the size and shape of the red circle.
    The diameter property will be about 67.

When you have finished your editor and outline panel should look like this:

We now need a factory to spawn our target.

  1. In the assets panel, right click the 'main' folder and select, 'New...', 'Factory'.
  2. Name the factory, 'target' and click, 'Create Factory'.

Again, thinking ahead, we are creating this factory outside of the collections so that we can easily use it in all the level collections.

  1. In the properties panel, set the 'Prototype' property to, '/main/target.go'.
  2. Save the changes by pressing CTRL-S or 'File', 'Save All'.
